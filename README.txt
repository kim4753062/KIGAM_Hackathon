-- 파일 구성 ====================================================================================
1. Mission0_EDA.py
3W dataset에 대한 전체적인 통계값 분석 및 비이상적인 압력값 처리를 위한 logical boundary 캐시파일을 './cached'폴더에 반환합니다.

2. Misison1_classification_instances_RF.ipynb [1차 모형]
전처리된 데이터를 기반으로 RF 모형으로 인스턴스별 클래스를 분류합니다.

3. Mission2_binary_classification_with_AE.ipynb [2차 모형]
LSTM 오토인코더를 학습하고, 이를 활용하여 우수한 데이터를 선별 후 LSTM 분류기를 학습하여 클래스별로 정상과 비이상을 분류합니다.

4. models.py
LSTM 모형의 구조 및 학습에 필요한 함수들을 포함한 모듈입니다.

5. module.py
데이터 전처리와 부가적인 기능들을 가진 함수들를 포함한 모듈입니다.

-- 디렉토리 구성 ==================================================================================
1. cache
학습된 기계학습 모형의 가중치를 저장하는 폴더입니다. (캐싱 포멧 .pth)
2. cached
EDA를 통해 구해진 압력의 logical boundary을 저장하는 폴더입니다. (캐싱 포멧 .pkl)
3. data
0번부터 8번까지 클래스별로 하위폴더가 존재하고, 하위폴더에는 해당 클래스에 해당하는 csv 파일이 존재합니다.
4. fig
코드를 실행하며 얻은 사진을 저장하는 폴더입니다. 하위폴더로는 Mission1과 Mission2가 존재합니다.
